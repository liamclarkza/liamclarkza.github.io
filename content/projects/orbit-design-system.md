Orbit is an in-progress design-system project exploring how a small set of purposeful tokens can produce interfaces that feel both quiet and capable.

## The direction

The system favours soft surfaces, restrained borders, compact controls, and subtle interaction states. Its light and dark themes share the same underlying hierarchy, rather than behaving like two unrelated visual systems.

## What I’m exploring

- A practical token model for colour, spacing, radius, and motion
- Components that remain legible without feeling visually heavy
- Patterns that work consistently across light and dark environments

More detail and a public link can be added here as the project develops.
